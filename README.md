# CI-GNN — 镍顶吹炉故障诊断系统

基于 CI-GNN（Granger因果启发的图神经网络）的镍顶吹炉多传感器故障诊断，
具备内置可解释性，能指出导致每类故障的关键传感器。

---

## 系统设计

### 变量定义（20个）

| 编号 | 变量 | 物理含义 | 正常范围 |
|------|------|---------|---------|
| X1-X3 | T1-T3 | 炉内温度区域1-3 | 1200-1350 ℃ |
| X4-X6 | P1-P3 | 炉压/喷枪压/供氧压 | 120-180 kPa |
| X7-X9 | F1-F3 | 氧气/冷却水/氮气流量 | 各不同 |
| X10-X12 | V1-V3 | 振动/声学传感器 | 0.5-2.0 mm/s |
| X13-X15 | CO/CO₂/SO₂ | 烟气成分 | 工艺范围 |
| X16-X18 | T_炉口/T_炉壳/喷枪位置 | 炉口及外壳监测 | 各不同 |
| L1, L2 | 潜在变量 | 炉况综合指数/设备磨损指数 | 归一化 |

### 六类标签

| 类别 | 名称 | 主要异常变量 |
|------|------|------------|
| 0 | 正常运行 | — |
| 1 | 炉衬侵蚀/穿炉 | T1-T3↑↑, T_炉口↑, L1漂移 |
| 2 | 喷枪堵塞/烧损 | P2异常, F1↓, V1-V2↑ |
| 3 | 供氧异常(过吹) | F1↑↑, CO₂↑, CO↓ |
| 4 | 炉口积渣/喷溅 | T_炉口↑↑, V1-V3↑↑↑, CO↑ |
| 5 | 冷却系统异常 | F2(冷却水)↓↓, T_炉壳↑↑ |

### 数据集规模

| 类别 | 样本数 |
|------|-------|
| 正常 | 4000 |
| 故障1-5 | 各800 |
| **总计** | **8000** |
| 划分 | 80/10/10 (分层) |

## 图结构设计

```
20个传感器节点，每节点8维特征:
  [均值, 标准差, 最大值, 最小值, 极差, 偏度, 峰度, RMS]

边连接：
  ① 物理先验边：基于镍顶吹炉工艺因果关系
     供氧压P3 → 氧流量F1 → 炉温T1-T3 → 烟气成分
     冷却水F2 → 炉壳温度 → 炉温
     炉温 → 炉口温度 → 振动
  ② 数据驱动边：皮尔逊相关系数 > 0.5 的变量对
```

## 安装与运行

```bash
pip install torch torch_geometric scipy scikit-learn matplotlib numpy
```

```bash
# 默认运行（自动生成数据）
python main.py

# 强制重新生成数据
python main.py --force_regen

# 自定义参数
python main.py \
  --GIN_hidden_dim 128 \
  --GIN_num_layers 4 \
  --stage2_epochs 300 \
  --batch_size 128
```

## 输出

运行结束后生成：

| 文件 | 内容 |
|------|------|
| `ci_gnn_nickel.pt` | 训练好的模型权重 |
| `fault_importance.png` | 各故障关键传感器热力图 |
| `confusion_matrix.png` | 混淆矩阵 |

### 可解释性示例输出

```
类别 1: 炉衬侵蚀/穿炉
  传感器             重要性分数
  T3(炉温区3)           0.8921
  T2(炉温区2)           0.8534
  T_炉口               0.7823
  T1(炉温区1)           0.7412
  L1(炉况)              0.6234
```
CI-GNN 通过因果子图自动定位到温度相关传感器，
与该故障的物理机制（炉衬损坏→热量外散→温度异常）完全吻合。

## 文件结构

| 文件 | 说明 |
|------|------|
| `main.py` | 训练入口 |
| `data_generator.py` | 仿真传感器数据生成 |
| `graph_builder.py` | 时序特征 → 图结构转换 |
| `utils.py` | 数据加载 + Acc/F1/MCC评估 |
| `explain.py` | 因果子图可解释性分析 + 可视化 |
| `GraphVAE.py` | 图变分自编码器 |
| `GIN_classifier.py` | GIN图分类器 |
| `GCE.py` | 生成式因果解释器（核心） |
| `causaleffect.py` | CMI估计 + HSIC独立性约束 |
